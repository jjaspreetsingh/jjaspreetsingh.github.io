# Jaspreet Singh - Portfolio Website

A minimalist portfolio website showcasing INTP analytical skills and technical projects.

## Features

- Light/Dark theme toggle
- Responsive design
- Clean, minimalist aesthetic
- Fast loading
- Easy to customize

## Technologies Used

- HTML5
- CSS3 (CSS Variables)
- JavaScript (ES6+)
- PHP

## Setup

1. Upload all files to your web server
2. Update the `config.php` with your information
3. Customize content in each page
4. Test the theme toggle functionality

## Customization

- Update colors in CSS variables
- Modify content in PHP files
- Add your projects and skills
- Update social media links
